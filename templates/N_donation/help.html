Vailo.ai is an intelligent automation platform using what’s app. Reach your business everywhere using what’s app. Vailo.ai platform is a pioneer in bringing next generation experience to day today commerce. By leveraging Vailo.ai businesses have easy way of converting their social media foot falls into sales into direct sales. Businesses have direct access to their own customer data for future engagements. Businesses can have a truly direct commerce channel with customers. Vailo.ai solution runs on a fully secured platform along with end to end encryption.

Vailo.ai is the idea behind making Direct Commerce a reality. It helps to automate Business processes, workflow and tasks. You can easily customize your data through the vailo.ai Dashboard. There are numerous solutions accessible right now that deal with the idea of direct commerce in bits and pieces, but it is obvious that there is a lack of a workable solution that deals with all elements. Vailo.ai offers software as a service that is reasonably priced and easy to acquire, launch, and use. In the initial roadmap, we are releasing a solution that gives you a head start on your Direct Commerce journey. It has a number of innovative feature additions to keep you current and competitive in the Direct Commerce industry as a whole.

Vailo.ai main area of interest is using the most recent technology to address problems in the real world in an easy and economical way. With Vailo.ai business can manage repetitive task such as Sourcing, Screening and Scheduling. Enhance your business with the power of what’s app and Telegram Bring next generation customer experience to your business with our unique solution.

When customer send ‘Hi’ to register what’s app number of Business to book ticket they will receive A1-Image, A2-header, A3-footer, A4 to A6-Button Name. Welcome message customised by business using dashboard (Refer serial number→5.2.1 → 1.1). For example customer will get

After getting this page customer will select the option like

  • Book Ticket: If the customers visiting 1st time they will book event ticket by clicking Book Ticket.
  • My Ticket: Here customers booked ticket will available to see.They can use or transfer the event ticket.
  • Cancel Booking: Customers can cancel the ticket.
3.1.1 Ticket booking by customer through what’s app:

After clicking Book ticket, customers will get Event List. If business setup more than 1 Event using dashboard (Refer serial number→5.2.2→level 1). customer will get B1-Image, B2-Message, B3-footer, B4-button to select particular Event in what’s app.

Business setup Events using dashboard (Refer serial number→5.2.2→Level 1). customer will get Event list to select particular Event in what’s app

  • Note : If the business set up only one event customer will not get option to select the particular event.
3.1.3 Select particular location

A) After selecting Event name, Customer has to select location set up by business using vailo.ai dashboard (Refer serial number→5.2.2→Level 2). customer will get D1-Image, D2-header, D3-message, D4-footer, D5-button to select location

B) After clicking select location customer will get the location list set up by the business vailo.ai dashboard (Refer serial number→5.2.2→level 2). customer will get D1-Image, D2-header, D3-message, D4-footer, D5-button to select location

3.1.4 Select particular ticket category

A)After selecting Event name, Event location Customer has to select ticket category set up by business using vailo.ai dashboard (Refer serial number→5.2.2→level 3) customer will get Image, message, header, footer, button to select slots

B) After clicking select Event slots customer will get the slots list set up by the business vailo.ai dashboard (Refer serial number→5.2.2→level 2).

3.1.5 Select number of ticket

A) After selecting ticket category Customer has to select the number of Tickets that they would like to book to watch the event set up by business using Dashboard

B) After clicking select number of ticket customer will get the list set up by the business vailo.ai dashboard

C) If all ticket sold out but some customers trying to book the event ticket they will get N4-Image, N5-message (Refer serial number→5.2.1→1.5→B).

3.1.6 Payment process

A) After selecting number of ticket customer has to make payment for generate an event ticket set up by business using vailo.ai Dashboard (Refer serial number→5.2.1→1.3→E to H)..

B) After clicking Pay button customer will get the payment details set up by the business in Razor pay (Refer serial number→4.4.2→level 2)

D) After filling phone number customer will get the option to select the payment method which is comfortable to customer.

After filling phone number customer will get the option to select the payment method which is comfortable to customer.

3.1.7 confirmation message

After completing payment process customer will get confirmation message, generated Ticket with QR CODE set up by Business using Dashboard(Refer serial number→5.2.1→1.3→A).

3.2 My Ticket
3.2.1 1 View ticket

If the customers book two tickets they can view both ticket and also they can view their all previous booked ticket here. Set up by business using Dashboard (Refer serial number→5.2.1→1.4→A).

Customer will get the option to “Use” OR “transfer”. If customers want to see event they can use the ticket, If customers wants to transfer their generated ticket to someone they can transfer their ticket. Set up by business using Dashboard (Refer serial number→5.2.1→1.5→A).

If the Customer wants to cancel the generated ticket they have to select cancel booking they will get U1-Image, U2-Message set up by business using vailo.ai dashboard (Refer serial number→5.2.1→1.4→B).

6 Scanner

Before using scanner New Business have to create your account. Visit https://vailo.ai/ Main use of scanner is to verify the generated Ticket. It will save the Time and energy.

After scanning the ticket, Scanner gives you 3 Result

  • Allowed –when 1st user scan the ticket QR CODE they get allowed notification.
  • Not Allowed – when Ticket QR CODE used by different person they get notification.
  • In valid – When someone use Different QR CODE they will get this notification
